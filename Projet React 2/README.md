# 📝 ToDo App React (Vite)

Application de gestion de tâches développée avec **React (Vite)** dans le cadre d’un exercice frontend.

---

## 🚀 Objectif du projet

Créer une application permettant de :

* Ajouter des tâches
* Afficher la liste des tâches
* Marquer une tâche comme complétée
* Supprimer une tâche

---

## 🛠️ Technologies utilisées

* React (Hooks : useState, useEffect)
* Vite (outil de build rapide)
* JavaScript (ES6+)
* CSS3
* LocalStorage (sauvegarde des données)

---

## ⚙️ Installation du projet

### 1. Cloner le projet

```bash
git clone https://github.com/ton-utilisateur/todoapp-react.git
```

### 2. Aller dans le dossier

```bash
cd todoapp-react
```

### 3. Installer les dépendances

```bash
npm install
```

### 4. Lancer le projet

```bash
npm run dev
```

Puis ouvrir :

```
http://localhost:5173
```

## ✨ Fonctionnalités

### ➕ Ajouter une tâche

Permet d’ajouter une nouvelle tâche via un formulaire.

### 📋 Afficher les tâches

Toutes les tâches sont affichées dynamiquement.

### ✔️ Compléter une tâche

Un clic sur une tâche permet de la marquer comme terminée.

### ❌ Supprimer une tâche

Possibilité de supprimer une tâche définitivement.

### 💾 Sauvegarde automatique

Les tâches sont sauvegardées dans le **localStorage**.
