# 📇 Gestionnaire de Contacts - React

## 📌 Description

Ce projet est une application web développée avec React permettant de gérer une liste de contacts.

L’utilisateur peut :
- ajouter un contact
- afficher la liste des contacts
- modifier un contact
- supprimer un contact

Le projet a été réalisé dans le cadre d’un exercice pédagogique autour de React et de la gestion d’état avec les hooks.

---

## 🚀 Fonctionnalités

### ✅ Gestion des contacts
- Ajout d’un contact
- Affichage dynamique de la liste
- Modification d’un contact
- Suppression d’un contact

---

### 🔎 Fonctionnalités bonus
- Recherche de contacts
- Tri des contacts par nom
- Validation des champs du formulaire
- Vérification du format email

---

## 🛠️ Technologies utilisées

- React
- Vite
- JavaScript (ES6+)
- CSS
