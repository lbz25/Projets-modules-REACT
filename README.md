#  Projets React - Module - LIBIZANGOMO Bachelor 3 DAD

Ce dépôt contient tous les projets réalisés durant le module React.

## Projets

### Projet 1 - Todo App

Créer une application React permettant :
- d’ajouter des tâches,
- d’afficher la liste des tâches,
- de marquer une tâche comme complétée,
- et de supprimer une tâche.

### Projet 2 - Contact App

Créer une application React permettant :
- d’ajouter un contact avec prénom, nom, email et téléphone
- d’afficher la liste des contacts
- de modifier un contact
- de supprimer un contact.

### Projet 3 - Blog App

Créer une application React multi-pages permettant à un
utilisateur de s'inscrire, se connecter, consulter, créer, modifier et
supprimer des articles.
L'accès à certaines pages doit être protégé par authentification.



## 🚀 Objectif
Centraliser tous les exercices et projets React du cours dans un seul dépôt organisé.
